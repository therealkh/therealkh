### What I care about

- **Architecture that enforces itself** — FSD with ESLint boundaries and dependency-cruiser, not just folder conventions
- **Animations on the UI thread** — Reanimated worklets, shared values, gesture-driven interactions at 60fps
- **Type safety that actually helps** — overloaded generics, codegen from OpenAPI/GraphQL, minimal type assertions
- **Security beyond the defaults** — Keychain/Keystore, SHA-256 PIN crypto, biometric auth, app lock systems
